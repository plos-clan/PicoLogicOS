
# Pico Logic OS 模块

- `loader`: 引导
- `kernel`: 内核
- 基本库
  - `define`: 基本宏和类型头文件
  - `base`: 基本库
  - `algorithm`: 算法库
  - `data-structure`: 数据结构库
  - `libc`: C 标准库 和 C 标准数学库
  - `libc++`: C++ 标准库
  - `misc`: 杂项库
- 渲染
  - `pl2d`: 2D 图形库
  - `pl3d`: 3D 图形库
  - `plui`: UI 库
- 多媒体
  - `plac`: 音频压缩库
  - `plic`: 图片压缩库
  - `plvc`: 视频压缩库
- `apps`: 应用程序
